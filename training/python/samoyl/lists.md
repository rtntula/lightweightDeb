Список -- упорядоченный изменяемый тип данных.

```
In [1]: list1 = [1,2,'four','1, 2, 3']

In [3]: id(list1)
Out[3]: 2124688232

In [4]: list1[0] = 1000

In [5]: list1
Out[5]: [1000, 2, 'four', '1, 2, 3']

In [6]: id(list1)
Out[6]: 2124688232

In [13]: list2 = list1

In [14]: list2
Out[14]: [7000, 2, 'four', '1, 2, 3']

In [15]: list2[0] = 9000

In [16]: list1
Out[16]: [9000, 2, 'four', '1, 2, 3']
```
